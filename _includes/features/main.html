How to use?

CDN for GitHub files

{{site.cdn.url}}/gh/:user/:repo/:tag/:file

CDN for GitLab files

{{site.cdn.url}}/gl/:user/:repo/:tag/:file

CDN for Bitbucket files

{{site.cdn.url}}/bb/:user/:repo/:tag/:file

CDN for custom endpoint images

{{site.cdn.url}}/img/:image_url

CDN for any popular web library

{{site.cdn.url}}/libs/:name/:version/:file

CDN for WordPress static files

{{site.cdn.url}}/wp/c/:version/wp-includes/:file {{site.cdn.url}}/wp/p/:plugin_name/:version/:file {{site.cdn.url}}/wp/t/:theme_name/:version/:file

Get a favicon from a domain

{{site.cdn.url}}/favicons/:domain

Take a screenshot of a webpage

{{site.cdn.url}}/screenshot/:url
